## FCR_D_UP band width
The band width $β^{FCR\_D\_UP}$ used for the regulation limit of FCR_D for up-regulation, can be set by the commands:
```
Command: set fcr_d_band <value>
```

If commands not set: The value $β^{FCR\_D\_UP} = 0.4$ will be used.
